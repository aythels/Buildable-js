• Every outer module can be dragged and resized in any way.
  • Dragging a parent module moves all its child modules.
  • Dragging a child module unmounts it.
  • Dragging a module over the top or bottom border of another module mounts it.
  • Resizing a parent module resizes all child modules (buggy).
  • Resizing the height of a child module resizes its siblings.
  • Currently, only vertical snapping is implemented.
